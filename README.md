# What is Goobvale?:
Goobvale is a minecraft-style sandbox game thats made entirely in python with only text inputs

in goobvale, you can go find things like trees, lakes, rock, you can also mine, craft things, use items or even find secrets, you can also get pets and other special item to make your experience even better, so why not play goobvale, its very simple to play and theres lots of features

one thing to note is that you need to remember to pick up everything, it never auto pick ups
